پیش‌نیازها و کتابخانه‌ها
در این پروژه **فقط از کتابخانه‌های استاندارد پایتون** استفاده شده و منطق پارس کردن خطوط لاگ کاملاً به صورت دستی و با Regex مدیریت شده است:
- `re`: جهت پارس کردن دقیق خطوط لاگ با الگوهای بهینه.
- `argparse`: جهت مدیریت پیشرفته و استاندارد آرگومان‌های ورودی CLI.
- `json`: جهت خروجی گرفتن ساختاریافته از گزارش‌ها.
- `gzip`: جهت پشتیبانی نیتیو از فایل‌های فشرده لاگ.


راهنما اجرا:
برای اجرا این برنامه کافی است دستور زیر را در ترمینال وارد کنید (مسیر فایل لاگ های خود با به جای 'access.log' قرار دهید)
python cli_analyzer.py access.log

تنظیم تعداد آدرس های پر تکرار:
به صورت دیفالت 10 است
python cli_analyzer.py access.log --top 5

تنظیم خروجی json:

python cli_analyzer.py access.log --json
